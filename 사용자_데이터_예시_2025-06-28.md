# 사용자 데이터 예시 (2025-06-28)

```json
[
  {
    "username": "mentor1",
    "password": "pass1",
    "name": "홍길동",
    "contact": "mentor1@email.com",
    "type": "mentor"
  },
  {
    "username": "mentee1",
    "password": "pass2",
    "name": "김철수",
    "contact": "010-1234-5678",
    "type": "mentee"
  }
]
```

- username: 아이디
- password: 비밀번호
- name: 이름
- contact: 연락처(이메일 또는 전화번호)
- type: "mentor" 또는 "mentee"
