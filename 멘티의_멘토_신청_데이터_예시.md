# 멘티의 멘토 신청 데이터 예시

```json
[
  {
    "mentee": "mentee1",
    "mentor": "mentor1",
    "status": "pending"
  },
  {
    "mentee": "mentee2",
    "mentor": "mentor1",
    "status": "accepted"
  },
  {
    "mentee": "mentee3",
    "mentor": "mentor2",
    "status": "rejected"
  }
]
```

- mentee: 신청한 멘티의 username(아이디)
- mentor: 신청 대상 멘토의 username(아이디)
- status: "pending", "accepted", "rejected" 중 하나
