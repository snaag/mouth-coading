# 상담 데이터 기록 예시

```json
[
  {
    "writer": "mentee1",
    "mentor": "mentor1",
    "date": "2025-06-28T14:30:00Z",
    "reply": "상담 답변 내용 예시입니다."
  },
  {
    "writer": "mentee2",
    "mentor": "mentor2",
    "date": "2025-06-27T10:00:00Z",
    "reply": "답변이 아직 없습니다."
  }
]
```

- writer: 상담을 작성한 사용자의 username(아이디)
- mentor: 상담 대상 멘토의 username(아이디)
- date: 작성 날짜 (ISO 8601 형식)
- reply: 상담 답변 내용 (없으면 빈 문자열 또는 안내 문구)
